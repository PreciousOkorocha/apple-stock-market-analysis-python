# Apple (AAPL) Stock Market Analysis Using Python

## Project Overview

This project analyzes Apple's historical stock market data using Python to uncover trends, patterns, and insights from financial data.

The analysis focuses on data cleaning, transformation, time-series analysis, and feature engineering using historical stock market data from Yahoo Finance.

---

## Objectives

The main objectives of this project are:

- Clean and preprocess historical stock data.
- Apply advanced Pandas techniques for data transformation.
- Perform time-series analysis on stock performance.
- Engineer new financial features.
- Visualize stock trends and extract meaningful insights.

---

## Dataset

**Source:** Yahoo Finance - Apple Inc. (AAPL) Historical Data

The dataset contains:

- Date
- Open Price
- High Price
- Low Price
- Close Price
- Adjusted Close Price
- Trading Volume

The analysis covers five years of historical stock market data.

## Data Analysis Process

### 1. Data Cleaning

Performed:

- Missing value checks
- Duplicate removal
- Date formatting
- Data type corrections
- Sorting data chronologically

### 2. Data Transformation

Applied:

- Filtering
- Sorting
- Grouping
- Aggregation
- Monthly and yearly summaries

### 3. Feature Engineering

Created:

- Daily Price Change
- Percentage Price Change
- 7-Day Moving Average
- 30-Day Moving Average
- Monthly Returns
- Volatility

### 4. Time-Series Analysis

Analyzed:

- Closing price trends
- Trading volume movement
- Monthly performance
- Moving averages
- Stock volatility

---

## Key Insights

- Apple's stock price showed an overall upward trend during the five-year period.
- Moving averages helped identify long-term price direction.
- Trading volume changed significantly during periods of market activity.
- Monthly returns showed variations in stock performance over time.
- Volatility analysis highlighted periods of increased market uncertainty.
